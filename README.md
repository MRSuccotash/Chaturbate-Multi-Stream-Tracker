## Installation

1. Clone this repository or download as ZIP.
2. Open Chrome and go to `chrome://extensions/`.
3. Enable **Developer mode** (toggle in the top right).
4. Click **Load unpacked** and select the extension folder.

## How to Use

1. Browse Chaturbate as usual.
2. Click the **"Track"** button that appears below any model's room card.
3. Your personal multi-stream dashboard will appear at the top of the page!
